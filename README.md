## Prerequesets

```cmd
pip install psutil
pip install pypresence
pip install pywinauto
```

## How to run

**Simply double  click it, it will open in a hidden cmd**

```cmd
python "VBox Discord RPC.py"
```

**You can put it in the start-up folder if you want it to be ready always**
